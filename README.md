## 🛠️ Ejemplo de Referencia API

### Obtener todos los ítems
```http
GET /api/items
```
| Parámetro  | Tipo     | Descripción                  |
|------------|----------|------------------------------|
| `api_key`  | `string` | **Requerido**. Tu clave API. |

### Obtener un ítem
```http
GET /api/items/${id}
```
| Parámetro  | Tipo     | Descripción                       |
|------------|----------|-----------------------------------|
| `id`       | `string` | **Requerido**. ID del ítem a buscar. |
